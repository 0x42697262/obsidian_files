# Setup

1. Get [Obsidian](https://obsidian.md/download) first.
2. Clone the repository

```
git clone https://github.com/0x42697262/obsidian_files.git
```

3. Import the cloned repository in Obsidian.
4. Check the [index](*Notes.md).
