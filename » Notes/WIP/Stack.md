# Stack

---

A [[Data Structure]] with contiguous block of memory containing data. Adding and removing data will always be performed at the last data placed[^1].

## Example

```
╔═══════════════╗
║       c       ║ ⟵ top of the stack (stack pointer)
╠═══════════════╣
║       b       ║
╠═══════════════╣
║       a       ║ ⟵ bottom of the stack
╚═══════════════╝
```
