# Datalink Codes

| Datalink Type            | Code              |
| ------------------------ | ----------------- |
| Reserved                 | 0 - 1000          |
| Un-encapsulated HCI (H1) | 1001              |
| HCI UART (H4)            | 1002              |
| HCI BSCP                 | 1003              |
| HCI Serial (H5)          | 1004              |
| Unassigned               | 1005 - 4294967295 |
