# GNU Debugger — set

---

## Disable [[Address Space Layout Randomization]] Inside [[GNU Debugger]]

```
(gdb) set disable-randomization <on/off>
```

## Set Assembly Syntax To Intel

```
(gdb) set disassembly-flavor intel
```
